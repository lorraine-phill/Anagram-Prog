Anagram-Prog
============

29-8-13

Create the following program using Ruby: You're given a list of strings, return a list of list of strings 
of anagrams, where each element of the returned list is a list of words that are anagrams among them.
For example input:  [stars, mary, rats, tars, army, banana] 
and output: [ [rats, tars], [army, mary], [stars], [banana] ].
